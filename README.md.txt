#  Netflix India Landing Page Clone

A fully responsive **Netflix India Landing Page Clone** built using **HTML, CSS, and JavaScript**.

This project replicates the UI design and FAQ interaction of the official Netflix homepage and is created as a portfolio project for practicing modern web design and DOM manipulation.

## Features

-  Responsive Netflix-style layout
-  Hero section with background image and overlay
-  Feature sections with images and videos
-  Floating label email input animation
-  Fully working FAQ accordion (just like real Netflix)
-  Mobile, Tablet, and Desktop responsive design
-  Clean and well-structured CSS
-  DOM manipulation using Vanilla JavaScript
